Release Summary

ClearByte v3.5.2 (2026014) marks the first production release of ClearByte and introduces important improvements in stability, performance optimization, and overall user experience across duplicate scanning, storage analysis, and cleanup workflows. Multiple refinements have been implemented to improve reliability, responsiveness, and operational consistency during real-world usage.

This release is focused on production-grade duplicate file scanning, storage analysis performance, folder selection reliability, cleanup workflow responsiveness, memory optimization, and improved handling of large file collections and deep directory structures.

Enhancements

* Improved duplicate scanning accuracy and reliability
* Faster scan execution and result generation
* Enhanced storage analysis performance
* Optimized folder selection and directory loading workflow
* Improved responsiveness during large-scale scans
* Refined user interface interactions and navigation
* Reduced memory consumption and improved resource management
* Better handling of large file collections and deep directory structures
